# AdvBookMaker

このソフトはなつかしのゲームブックの電子書籍版製作をサポートします。 
 
ゲームブックとは、本の形をしたアドベンチャーゲームです。たいては状況を示した本文 「君は薄暗い部屋の中にいる。目の前には一つだけ扉がある」等のような文章につづいて 「扉を開ける→４０ページに飛べ」等のように選択枝にページ数への参照が印刷され、その 指示に従ってページを移動することでコンピュータのアドベンチャーゲームを再現してい るものです。 ※詳しくは Wiki 等を参照して頂戴。 
 
かつては紙のページで選択枝を選んでいましたが、電子書籍にすることで、この「〇〇ペー ジへ飛べ」の部分をハイパーリンクにして、簡単にジャンプすることができるようになりま した。 とはいってもリンクの設定等が面倒なのですね。でも安心、このソフトを使えば、半自動的 にページの設定が行え、なんと epub ファイルまで自動でつくってくれます
